개인정보 처리방침

DevRoutine (이하 "앱")은 사용자의 개인정보를 소중히 여기며, 다음과 같이 처리방침을 안내드립니다.

최종 업데이트: 2024년 12월 18일

1. 수집하는 정보

① 자동 수집 정보

  • 앱 사용 데이터 (익명 통계)
  • 광고 ID (IDFA/GAID)
  • 기기 정보 (운영체제, 앱 버전, 기기 유형 등)

② 사용자가 입력하는 정보

  • 루틴 제목, 메모, 우선순위
  • 루틴 완료 기록
  • 앱 설정 정보

2. 로컬 저장소 사용

루틴 데이터는 Hive 데이터베이스를 통해 사용자 기기 내에만 저장되며 서버로 전송되지 않습니다. 앱 삭제 시 모든 데이터는 완전히 삭제됩니다.

3. 알림 권한

  • Android: POST_NOTIFICATIONS 권한
  • iOS: 푸시 알림 권한

사용 목적: 루틴 리마인더, 챌린지 알림, 앱 업데이트 알림

4. 광고 및 분석

Google AdMob

  • 광고 수익을 통한 무료 서비스 제공
  • 수집 정보: 광고 ID, 대략적인 위치
  • 사용자 설정에서 맞춤형 광고 해제 가능

Firebase Analytics 및 Crashlytics

  • 익명 통계 및 오류 수집
  • 개인정보는 포함되지 않음

5. 개인정보 보호 및 보안

  • 기기 내부 데이터는 암호화 저장
  • HTTPS 통신 암호화 사용
  • 정기적인 보안 업데이트 수행

보관 기간

  • 로컬 데이터: 사용자가 삭제 전까지
  • 분석 데이터: 최대 26개월
  • 광고 데이터: Google 정책에 따름

6. 제3자 서비스

  • Google AdMob
  • Firebase (Analytics, Crashlytics)

각 서비스의 개인정보처리방침:

  • Google 개인정보처리방침
  • Firebase 개인정보처리방침

7. 아동의 개인정보

본 앱은 13세 미만 아동을 대상으로 하지 않으며, 고의로 개인정보를 수집하지 않습니다.

8. 위치 정보

위치 정보는 수집하거나 저장하지 않습니다.

9. 개인정보 처리방침 변경

  • 앱 업데이트 시 새로운 정책 공지
  • 중대한 변경 시 별도 알림 제공

10. 사용자 권리

  • 개인정보 열람 및 수정 요청
  • 개인정보 삭제 요청 (앱 삭제 시 가능)
  • 개인정보 처리 중단 요청
